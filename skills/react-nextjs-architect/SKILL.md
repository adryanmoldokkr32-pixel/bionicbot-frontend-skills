# React/Next.js Architect

This skill enables bionicbot to architect robust, scalable, and SEO-friendly web applications using the Next.js framework.

## Instructions

1. Choose between Page Router or App Router based on requirements.
2. Use Server Components for better performance and SEO.
3. Implement Client Components only when necessary for interactivity.
4. Use file-based routing for clean architecture.
5. Optimize with Next.js features like `Image`, `Link`, and `Script` components.

- "Build a server-side rendered dashboard page for user data."
