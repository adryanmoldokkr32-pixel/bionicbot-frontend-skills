# Browser Compatibility Tester

This skill checks and fixes issues specific to different browser engines (Chrome, Safari, Firefox, Edge).

## Instructions

1. Identify CSS properties with poor browser support using 'Can I Use'.
2. Add vendor prefixes when necessary.
3. Use polyfills for missing modern JavaScript features.
4. Test rendering issues specific to iOS Safari.
5. Provide fallback styles for older browsers.

## Examples

- "Check if `backdrop-filter` is supported in older Firefox versions."
- "Fix a layout bug that only appears in Safari on iPhone."
