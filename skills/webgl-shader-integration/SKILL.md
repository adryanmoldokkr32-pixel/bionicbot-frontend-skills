# WebGL Shader Integration

This skill uses GLSL shaders to create hyper-realistic visual textures and effects for high-end web interfaces.

## Instructions
1. Write fragment and vertex shaders for specific UI elements.
2. Integrate Three.js or React Three Fiber for 3D scene management.
3. Optimize GPU usage to maintain 60 FPS on mobile devices.
4. Use shaders for background noise, liquid metals, and light glows.
