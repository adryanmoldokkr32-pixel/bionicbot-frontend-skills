# Micro-Frontend Orchestrator

This skill enables bionicbot to split a massive web application into smaller, independently deployable pieces that work together as one.

## Instructions
1. Define module boundaries (e.g., Auth, Dashboard, Market).
2. Implement a 'Shell' container using Module Federation.
3. Ensure shared state consistency across different micro-apps.
4. Optimize for independent scaling and fast builds.
