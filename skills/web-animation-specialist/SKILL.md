# Web Animation Specialist

This skill focuses on the technical implementation of smooth, physics-based animations for the web.

## Instructions

1. Define animation triggers (Appear, WhileInView, Exit).
2. Set spring-based physics for natural motion.
3. Orchestrate complex sequences using `staggerChildren` or timelines.
4. Use layout animations for smooth transitions when elements move.
5. Ensure animations don't negatively impact performance (GPU acceleration).

## Examples

- "Create a staggered list entrance animation with Framer Motion."
- "Animate a progress bar fill based on user scroll."
