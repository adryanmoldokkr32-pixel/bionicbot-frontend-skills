# Frontend Performance Tuner

This skill focuses on making websites load instantly and respond smoothly to user input.

## Instructions

1. Minimize Largest Contentful Paint (LCP) by optimizing hero assets.
2. Reduce Cumulative Layout Shift (CLS) by setting dimensions on media.
3. Minimize Third-Party scripts and bloat.
4. Implement code splitting and lazy loading for routes and images.
5. Compress and bundle assets (JS, CSS) for production.

## Examples

- "Analyze this page and suggest ways to improve its LCP score."
- "Optimize this React app to reduce the initial bundle size."
