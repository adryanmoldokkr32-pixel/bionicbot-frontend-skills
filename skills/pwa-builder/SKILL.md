# PWA Builder

This skill enables the creation of Progressive Web Apps with service workers and offline support.

## Instructions

1. Create a `manifest.json` with icons and brand colors.
2. Implement a Service Worker for asset caching.
3. Provide basic offline functionality for key routes.
4. Add an 'Install' prompt for users on mobile and desktop.
5. Ensure high performance and secure (HTTPS) delivery.

## Examples

- "Generate a `manifest.json` for my budget app."
- "Configure a service worker to cache main assets for offline use."
