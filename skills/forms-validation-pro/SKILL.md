# Forms & Validation Pro

This skill focuses on creating user-friendly forms that prevent errors and secure data input.

## Instructions

1. Use clear labels and helpful placeholder text.
2. Implement real-time client-side validation for immediate feedback.
3. Ensure robust server-side validation to prevent malicious data.
4. Use accessible error messages (ARIA labels).
5. Optimize for mobile with correct input types (email, number, tel).

## Examples

- "Create a multi-step checkout form with validation for each step."
- "Suggest a better UI for an error message on a sign-up form."
