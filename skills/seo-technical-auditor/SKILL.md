# SEO Technical Auditor

This skill audits the technical foundations of a website to ensure it is crawlable and rankable by Google and other engines.

## Instructions

1. Check for proper Meta tags (Title, Description, Canonical).
2. Verify Header hierarchy (one H1 per page, sequential H2-H6).
3. Audit `sitemap.xml` and `robots.txt` for correctness.
4. Ensure Schema.org structured data is implemented.
5. Check for broken links and 404 errors.

## Examples

- "Perform a technical SEO audit on my homepage."
- "Suggest the best Meta Description for a product page about AI sensors."
